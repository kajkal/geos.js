# point

Creates a GEOS point geometry from a position.

## Parameters

| Name | Type | Description |
| --- | :-: | --- |
| `pt` | [**`Position`**](https://datatracker.ietf.org/doc/html/rfc7946#section-3.1.1) | Point coordinates |

## Returns

[**`Geometry`**](../types/Geometry/index.mdx) A new point Geometry object

## Examples

```js
const geom = point([ 1, 1 ]);
toWKT(geom); // 'POINT (1 1)'
```
