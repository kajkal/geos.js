# lineString

Creates a GEOS line string geometry from an array of positions.
Line string must contain at least 2 positions.
Empty line strings with 0 positions are allowed.

## Parameters

| Name | Type | Description |
| --- | :-: | --- |
| `pts` | [**`Position`**](https://datatracker.ietf.org/doc/html/rfc7946#section-3.1.1)`[]` | LineString coordinates |

## Returns

[**`Geometry`**](../types/Geometry/index.mdx) A new Geometry object

## Throws

- [**`InvalidGeoJSONError`**](../types/errors.mdx#InvalidGeoJSONError) on line with 1 position

## Examples

```js
const geom = lineString([ [ 0, 0 ], [ 1, 1 ], [ 2, 2 ] ]);
toWKT(geom); // 'LINESTRING (0 0, 1 1, 2 2)'
```
