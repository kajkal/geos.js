# bounds

Calculates the bounds (also named bbox - bounding box or extent) of the geometry.
The bounds are the minimum rectangle that contains the entire geometry.

## Parameters

| Name | Type | Description |
| --- | :-: | --- |
| `geometry` | [**`Geometry`**](../types/Geometry/index.mdx) | The geometry for which the bounds are calculated |

## Returns

**`[`** **`number`**, **`number`**, **`number`**, **`number`** **`]`** An array of four numbers `[ xMin, yMin, xMax, yMax ]`

## Throws

- [**`GEOSError`**](../types/errors.mdx#GEOSError) when called on an empty geometry

## Examples

```js live noInline
const pt = point([ 3, 1 ]);
const ptExtent = bounds(pt); // [ 3, 1, 3, 1 ]
const poly = polygon([ [ [ 3, 3 ], [ 9, 4 ], [ 5, 1 ], [ 3, 3 ] ] ]);
const polyExtent = bounds(poly); // [ 3, 1, 9, 4 ]

// this example is alive!
render(preview({ pt, ptExtent, poly, polyExtent }));
```
