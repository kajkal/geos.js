# area

Calculates the area of a geometry.

Areal geometries have a non-zero area.
Others return 0.

## Parameters

| Name | Type | Description |
| --- | :-: | --- |
| `geometry` | [**`Geometry`**](../types/Geometry/index.mdx) | The geometry for which the area is calculated |

## Returns

**`number`** The area of the geometry

## Examples

```js live
const pt = point([ 3, 1 ]);
const ptArea = area(pt); // 0
const line = lineString([ [ 8, 1 ], [ 9, 1 ] ]);
const lineArea = area(pt); // 0
const poly = polygon([ [ [ 3, 3 ], [ 9, 4 ], [ 5, 1 ], [ 3, 3 ] ] ]);
const polyArea = area(poly); // 7
```
