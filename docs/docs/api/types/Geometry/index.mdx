# Geometry

Class representing a GEOS geometry that exists in the Wasm memory.

## Instance properties

These properties are defined on `Geometry.prototype` and shared by all `Geometry` instances.

### `Geometry.prototype.detached?` {#detached}

<p class='margin-bottom--sm margin-left--md'>Type: **`boolean`**</p>
<p class='margin-left--md'>Geometry can become detached when passed to a function that consumes
it, for example [`geometryCollection`](../../helpers/geometryCollection.mdx), or when manually
[freed](free.mdx). Although this Geometry object still exists, the GEOS
object that it used to represent no longer exists.</p>

## Instance methods

### [`Geometry.prototype.type`](type.mdx) {#type}

<p class='margin-left--md'>Returns geometry type.</p>

### [`Geometry.prototype.normalize`](normalize.mdx) {#normalize}

<p class='margin-left--md'>Organizes the elements, rings, and coordinate order of geometries in a consistent way, so that geometries that represent the same object can be easily compared.</p>

### [`Geometry.prototype.orientPolygons`](orientPolygons.mdx) {#orientPolygons}

<p class='margin-left--md'>Enforces a ring orientation on all polygonal elements in the input geometry.</p>

### [`Geometry.prototype.clone`](clone.mdx) {#clone}

<p class='margin-left--md'>Creates a deep copy of this Geometry object.</p>

### [`Geometry.prototype.toJSON`](toJSON.mdx) {#toJSON}

<p class='margin-left--md'>Converts the geometry to a GeoJSON geometry object.</p>

### [`Geometry.prototype.free`](free.mdx) {#free}

<p class='margin-left--md'>Frees the Wasm memory allocated for the GEOS geometry object.</p>
