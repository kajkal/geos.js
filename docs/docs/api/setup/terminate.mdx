# terminate

Terminates the initialized `geos.js` module and releases associated resources.

## Returns

[**`Promise`**](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise)\<**`void`**\> A Promise that resolves when the termination is complete
