# isEmpty

Returns whether the geometry is empty.
If the geometry or any component is non-empty, the geometry is non-empty.
An empty geometry has no boundary or interior.

## Parameters

| Name | Type | Description |
| --- | :-: | --- |
| `geometry` | [**`Geometry`**](../types/Geometry/index.mdx) | The geometry to check |

## Returns

**`boolean`** `true` when geometry is empty, `false` otherwise

## Examples

```js live noInline
const a = point([]);
const a_empty = isEmpty(a); // true
const b = point([ 0, 0 ]);
const b_empty = isEmpty(b); // false

// this example is alive!
render(preview({ a, a_empty, b, b_empty }));
```
